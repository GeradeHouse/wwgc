Download Certificate for Android

To enable your Android device to trust your local HTTPS server, you need to install a trusted CA certificate. This certificate is the root CA generated by mkcert and has been converted to DER format (rootCA.crt), which is required by Android.

Download Certificate (rootCA.crt)

Android Installation Instructions

  1. Download the Certificate: Tap the download button above to download the file "rootCA.crt".
  2. If the certificate is not automatically saved, transfer the "rootCA.crt" file to your Android device via USB, email, or cloud storage.
  3. On your Android device, open Settings and navigate to Security (or Biometrics & Security â†’ Encryption & Credentials on some versions).
  4. Select Install from storage (or a similar option) and locate the downloaded "rootCA.crt" file.
  5. When prompted to choose the type of certificate to install, ensure you select "Trusted CA certificate". (If the option is not available, it may mean that the certificate is missing required CA extensions. In that case, update mkcert to the latest version or configure OpenSSL accordingly.)
  6. After installation, restart your browser (or your device) to ensure the new certificate is recognized.

Once installed, your Android device will trust secure connections to https://192.168.31.18:8000/3d.html without any security warnings.
